By-time

Функциональность: случайным образом выбирает идею для досуга.

Внешний вид
4 экрана: Меню, Настройки, О программе, Генератор идеи.
На каждом экране присутствует ToolBar с пунктами: Меню, Настройки, О программе, Выход.

Экран "Меню"
Содержит 6 кнопок:
Один дома
Для компании
С ребенком(1-3 года)
С ребенком(3-6 лет)
С ребенком(старше 6 лет)
Что же почитать?

Экран "Настройки"
Позволяет изменять внешний вид программы.

Экран "О программе"
Содержит информацию о программе.

Экран "Генератор идеи"
Содержит кнопку "Старт" для генерации идеи и текстовое поле, в котором выводится идея.


Классы:
1.MainActivity
Создание Экрана "Меню".

2.SettingActivity
Создание Экрана "Настройки".

3.InfoActivity
Создание Экрана "О программе".

4.RandonGeneratorActivity
Создание Экрана "Генератор идеи"

5.RandomGenerator
Генерирует целое число, зная кол-во строк в таблице.
 
6.EmulatorDatabase
Временный класс для тестирования программы. Содержит кол-во строк, и объекты типа String.

7.Database
Обращается к базе данных.
По названию таблицы класс получает кол-во строк. 
По ключу в виде целого числа получает строку из таблицы.
